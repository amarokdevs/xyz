# **App Name**: File To HTML Converter

## Core Features:

- File Input: Allow users to select a file from their local machine using a standard file input element.
- HTML Generation: Generates an HTML file containing the Base64 encoded file data and the javascript logic necessary to re-construct the original file on download.
- Progress Display: Show a progress bar to indicate the encoding progress to the user, along with status messages.
- HTML Download: Allows the user to download the generated HTML file, containing the embedded file.

## Style Guidelines:

- Primary color: Subtle lavender (#E6E6FA) to evoke a sense of calm and seamlessness.
- Background color: Off-white (#F5F5F5) for a clean and modern aesthetic.
- Accent color: Pale blue (#ADD8E6) for buttons and progress bars, to draw attention without overwhelming the user.
- Font: 'Inter', a sans-serif font, to ensure readability and a contemporary feel throughout the interface.
- Center the file input, progress bar, and status message for a focused user experience.
- Use a smooth animation for the progress bar to provide visual feedback during the file processing.
